## 🧪 What To Do in This branch (IMPORTANT)

### ✅ Daily Practice Tasks

1. **Practice 5–10 Unix commands daily**

   * Run commands in terminal
   * Save command + output in markdown files

2. **Maintain a Daily Practice Log**

   * Update `daily-practice-log.md`
   * Mention:

     * Commands learned
     * Purpose
     * Example usage

3. **Git Practice**

   * Make **daily commits**
   * Write meaningful commit messages
   * Practice branching and merging

4. **GitHub Workflow**

   * Create branches
   * Raise pull requests
   * Resolve merge conflicts (practice)
